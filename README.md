# Fine-Tune

Repository for fine-tuning embedding models and implementing specialized RAG systems.

## Project Structure
- `nokia_rag/`: Core RAG logic and integration.
- `nokia-isam-bge-large/`: Fine-tuned BGE Large model weights.
- `Embedding_Model.ipynb`: Analysis and training notebooks.
- `requirements.txt`: Python dependencies.

## Setup
1. Configure your `.env` file with `HF_TOKEN` and other necessary environment variables.
2. Install dependencies: `pip install -r requirements.txt`.
